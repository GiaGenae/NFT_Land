NFT Land is an app created to house pieces of NFT art within their respective galleries. 

## Installation

Start this application by running:

#### Backend
1. `bundle` 
2. `rake db:create`
3. `rake db:migrate`
4. `rake db:seed`
5. `rails s -p 3001`
#### Frontend
6. `npm i`
7. `npm start`
6. The app will load in your browser and you can test it out!
